### Implementation Example
Using a linter tool like Snowflake's own linter or a third-party tool like SQLFluff, teams can create a configuration file that defines the naming conventions. For example:
```
# snowflake-linter-config.yml
rules:
  database_name:
    pattern: '^[a-z]{3,}_[a-zA-Z0-9_]+$'
```
