# User-Managed Virtual Warehouses for Tasks in Snowflake

**Description:**
This topic explains how to use user-managed virtual warehouses for tasks in Snowflake, allowing for more control over compute resources and task execution. User-managed virtual warehouses enable users to manage and optimize their own compute resources for tasks, improving performance and cost efficiency. By using user-managed virtual warehouses, users can better manage their tasks and reduce dependencies on Snowflake's auto-suspend and auto-resume features.
