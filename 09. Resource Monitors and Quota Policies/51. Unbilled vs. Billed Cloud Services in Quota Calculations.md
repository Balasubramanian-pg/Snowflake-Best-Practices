### Code Source Execution
```sql
CREATE VIEW daily_quota_usage AS
SELECT 
    DATE_TRUNC('day', usage_date) AS usage_day,
    SUM(unbilled_quantity) AS unbilled_quantity,
    SUM(billed_quantity) AS billed_quantity
FROM 
    snowflake.information_schema.usage_history
GROUP BY 
    usage_day;
```
