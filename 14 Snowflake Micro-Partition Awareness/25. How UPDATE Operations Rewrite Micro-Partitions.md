# UPDATE Operations Rewrite Micro-Partitions
UPDATE operations rewrite micro-partitions is a process in which the database management system rewrites the micro-partitions of a table when an UPDATE operation is performed. This process involves updating the existing micro-partitions and creating new ones if necessary. The goal of this process is to maintain the consistency and integrity of the data in the table. The UPDATE operations rewrite micro-partitions is an important aspect of database management as it ensures that the data is up-to-date and accurate.

## Implementation Example
```sql

    id INT,
    region VARCHAR(255),
    sales_date DATE,

    PARTITION p_2020 VALUES LESS THAN (2021),
```
