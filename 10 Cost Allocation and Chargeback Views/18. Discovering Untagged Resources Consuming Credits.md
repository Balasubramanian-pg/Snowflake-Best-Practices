### Code Source Execution
```sql
CREATE VIEW untagged_resources AS
SELECT 
    RESOURCE_NAME,
    RESOURCE_TYPE,
    CREDIT_USAGE
FROM 
    SNOWFLAKE.ACCOUNT_USAGE.WAREHOUSE_LOAD_HISTORY
WHERE 
    TAGS IS NULL OR TAGS = '';
```
