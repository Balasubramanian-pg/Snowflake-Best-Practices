      type: snowflake
      account: <account_name>
      user: <username>
      password: <password>
      database: <database_name>
      schema: <schema_name>
      warehouse: <warehouse_name>
      role: <role_name>

Then, in your dbt model file, you can specify the schema name using the 'schema' variable:

  )
