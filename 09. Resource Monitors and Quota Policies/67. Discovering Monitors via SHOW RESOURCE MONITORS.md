**Description:**
This topic explains how to discover and view existing resource monitors in Snowflake using the SHOW RESOURCE MONITORS command. Resource monitors track and alert on warehouse usage, helping to manage costs and optimize resource utilization. The SHOW RESOURCE MONITORS command provides a way to list and inspect the definitions and settings of existing monitors.
