# The SYSTEM$STREAM_HAS_DATA Function in Event Pipelines
The SYSTEM$STREAM_HAS_DATA function is used in event pipelines to check if a stream has data available for processing. By using this function, developers can create more efficient and scalable event pipelines that can handle large volumes of data.
