### Example Execution Logic Step-by-Step
When an INSERT operation is executed, Snowflake creates a new micro-partition to store the data. Micro-partitions are small, immutable chunks of data that Snowflake uses to store and manage data.

### Implementation Example
CREATE TABLE sensor_data (id INT, temperature FLOAT, timestamp TIMESTAMP); INSERT INTO sensor_data VALUES (1, 20.5, '2022-01-01 12:00:00'); INSERT INTO sensor_data VALUES (2, 21.0, '2022-01-01 12:01:00');
