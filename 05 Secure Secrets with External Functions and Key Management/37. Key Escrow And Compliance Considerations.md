# Dynamic Data Masking in Snowflake

**Description:**
Dynamic Data Masking (DDM) is a security feature in Snowflake that allows administrators to limit sensitive data exposure by masking it to unauthorized users. DDM applies masks to data at query time, ensuring that users only see authorized data. This feature helps prevent data breaches and ensures compliance with data protection regulations.
