### Code Source Execution
```sql
CREATE VIEW snowflake_credits_to_usd AS
SELECT 
    credit_history.date,
    credit_history.credits_used,
    credit_history.credits_used * exchange_rate.usd_per_credit AS usd_cost
FROM 
    snowflake.credit_history,
    (SELECT * FROM exchange_rate WHERE currency = 'USD') AS exchange_rate;
```

### Example Execution Logic Step-by-Step
This SQL view calculates the USD cost of Snowflake credits used over time. It joins the credit history with the exchange rate to get the USD value per credit.
