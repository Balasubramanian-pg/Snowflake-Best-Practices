# Assigning a Monitor to Multiple Virtual Warehouses in Snowflake

**Description:**
This topic explains how to assign a monitor to multiple virtual warehouses in Snowflake, allowing for efficient resource utilization and monitoring. By assigning a single monitor to multiple virtual warehouses, administrators can streamline monitoring and alerting processes. This feature is particularly useful for organizations with multiple virtual warehouses that require similar monitoring configurations.

## Short Explanation
**Overview:** Assigning a monitor to multiple virtual warehouses enables centralized monitoring and alerting for multiple resources.

**Problem Solved:** Manually configuring and monitoring multiple virtual warehouses can be time-consuming and prone to errors.

**Importance:** Efficient monitoring and resource utilization are crucial in analytics and warehousing environments.

**Use Cases:**
- Assigning a monitor to multiple virtual warehouses with similar configurations
- Streamlining monitoring and alerting for multiple resources

### Typical Scenario
A company has multiple virtual warehouses with similar monitoring requirements. Instead of creating separate monitors for each virtual warehouse, they can create a single monitor and assign it to all the virtual warehouses, ensuring efficient resource utilization and monitoring.

### Core Snowflake SQL Objects Used
`CREATE MONITOR`, `ALTER WAREHOUSE`

### Code Source Execution
```sql
CREATE MONITOR my_monitor FOR {warehouse_name} IN {WAREHOUSE_GROUP};
ALTER WAREHOUSE my_warehouse_1 SET MONITOR = my_monitor;
ALTER WAREHOUSE my_warehouse_2 SET MONITOR = my_monitor;
```

### Example Execution Logic Step-by-Step
To assign a monitor to multiple virtual warehouses, first create a monitor using the CREATE MONITOR statement. Then, use the ALTER WAREHOUSE statement to assign the monitor to each virtual warehouse.

### Implementation Example
CREATE MONITOR my_monitor FOR cpu_usage IN ('high' > 80);
ALTER WAREHOUSE my_warehouse_1 SET MONITOR = my_monitor;
ALTER WAREHOUSE my_warehouse_2 SET MONITOR = my_monitor;

### Explanation of the Code
- Step 1: Create a monitor named 'my_monitor' to track CPU usage.
- Step 2: Assign the 'my_monitor' to virtual warehouses 'my_warehouse_1' and 'my_warehouse_2'.
