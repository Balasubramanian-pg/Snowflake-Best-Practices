# Identifying Materialized Views in the Snowflake Catalog

**Description:**
This topic explains how to identify materialized views in the Snowflake catalog. Materialized views are pre-computed results that can improve query performance. The Snowflake catalog provides system views to identify and manage materialized views.

A business intelligence analyst wants to identify materialized views in the Snowflake catalog to optimize their data warehouse for better query performance.

### Core Snowflake SQL Objects Used
`CREATE MATERIALIZED VIEW`, `SHOW MATERIALIZED VIEWS`

### Code Source Execution
```sql
SHOW MATERIALIZED VIEWS IN DATABASE mydatabase;
```

### Example Execution Logic Step-by-Step
The SHOW MATERIALIZED VIEWS statement is used to list all materialized views in a specified database. The INFORMATION_SCHEMA.MVIEWS view can also be queried to retrieve information about materialized views.

### Implementation Example
CREATE MATERIALIZED VIEW my_mv AS SELECT * FROM mytable; SHOW MATERIALIZED VIEWS IN DATABASE mydatabase;
