### Example Execution Logic Step-by-Step
The build_data_dictionary script iterates through the INFORMATION_SCHEMA and extracts metadata about tables, columns, and data types. It then inserts this metadata into the data_dictionary table.

### Implementation Example

    CREATE SCRIPT build_data_dictionary()
    RETURNS => data_dictionary
    LANGUAGE SQL
    EXECUTE AS CALLER
    BEGIN
      DECLARE
        table_name VARCHAR;
        column_name VARCHAR;
        data_type VARCHAR;
        description VARCHAR;
      BEGIN
        FOR table_name IN SELECT table_name FROM INFORMATION_SCHEMA.TABLES
        LOOP
          FOR column_name IN SELECT column_name, data_type FROM INFORMATION_SCHEMA.COLUMNS WHERE table_name = table_name
          LOOP
            INSERT INTO data_dictionary (table_name, column_name, data_type, description)
            VALUES (table_name, column_name, data_type, 'No description available');
          END LOOP;
        END LOOP;
      END;
    END;
