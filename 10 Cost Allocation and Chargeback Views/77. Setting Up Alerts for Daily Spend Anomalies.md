A company wants to monitor its daily Snowflake spend to ensure that it stays within budget. They set up an alert to notify them when the daily spend exceeds a certain threshold or shows an unusual pattern.

### Core Snowflake SQL Objects Used
`CREATE TASK`, `CREATE ALERT`

### Code Source Execution
```sql

    CREATE TASK monitor_daily_spend
    WAREHOUSE = 'COMPUTE_WH'
    SCHEDULE = '0 8 * * *'
    COMMENT = 'Monitor daily spend'
    AS
    CALL monitor_spend();

    CREATE ALERT daily_spend_alert
    FOR {daily_spend > 1000}
    ON SCHEDULE 1 HOUR
    CALL notify_team();
  
```
