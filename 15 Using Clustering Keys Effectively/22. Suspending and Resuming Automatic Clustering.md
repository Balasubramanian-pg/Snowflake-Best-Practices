# Suspending and Resuming Automatic Clustering in Snowflake

**Description:**
Automatic clustering in Snowflake is a feature that re-clusters data in a table based on the defined clustering keys. Suspending and resuming automatic clustering allows for more control over when and how the re-clustering process occurs. This can be useful for optimizing performance and reducing costs.
