### Code Source Execution
```sql
CREATE VIEW cost_per_record_loaded AS
SELECT 
    SUM(credits_used) / COUNT(record_id) AS cost_per_record
FROM 
    data_loading_history;
```

### Example Execution Logic Step-by-Step
This SQL example creates a view that calculates the cost per record loaded by summing up the credits used for loading data and dividing it by the number of records loaded.
